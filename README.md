# 🚀 FixLedger

FixLedger is a frontend first civic impact platform that builds **trust through transparency**. It connects people around real world problems, enables collective action, and ensures funds are released only after verified proof of work.

🔗 Live Demo: https://fixledger-hackathon.netlify.app/


# 💡 Problem Statement

Modern civic and community problem-solving systems face three major gaps:

- **Isolation Barrier** → People facing the same issues are not connected  
- **Transparency Gap** → No clear visibility of fund usage and progress  
- **Trust Barrier** → No reliable proof that work has actually been completed  

FixLedger addresses these by making **action, proof, and money flow fully visible**.

⚙️ Tech Stack

- React (Vite)
- JavaScript (ES6+)
- CSS Modules / Plain CSS
- Local State Management
- Static Mock Data (No backend)
